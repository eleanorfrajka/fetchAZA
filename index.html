Welcome to fetchAZA’s documentation!

fetchAZA is a package to read the *.csv files generated by a Sonardyne Fetch AZA and parse the data into *.nc netcdf format.

For recommendations or bug reports, please visit https://github.com/eleanorfrajka/fetchAZA/issues/new
